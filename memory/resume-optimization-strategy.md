## 核心原则

不虚构经历，但用 JD 目标岗位的语言体系重新表述真实经历。

## 对标流程

1. 提取 JD 核心关键词（职责+要求）
2. 逐段对比简历 → 标出缺失关键词
3. 将真实经历用 JD 语言重写（保留数据、保留事实）
4. 每个模块都要呼应 JD 的至少一个关键词
5. 教育/背景等边缘信息也要往 JD 方向靠

## 腾讯多模态评测岗实战记录

原始定位：AIGC内容创作 / AI科技自媒体
目标定位：多模态模型评测项目管理 / 美学测评方向

关键映射：
- 园林设计专业 → 视觉审美训练（构图/光影/色彩/质感/风格/叙事/镜头语言）
- 10+人团队管理 → 项目全生命周期管理（需求拆解/排期/资源分配/风险管控）
- 70+工作流文档 → SOP文档+培训材料+质检规则
- 质量评分 6.8→8.5 → 多层级质检体系（抽检/交叉验证/专家复审/一致性校验）
- ComfyUI/SD 经验 → 熟悉多模态模型能力边界与 bad case
- Agent 开发 → vibe coding / 自动化 workflow 搭建
- novel-factory 审校流程 → 评测体系搭建思维
- B站内容 → 多模态模型评测内容输出
